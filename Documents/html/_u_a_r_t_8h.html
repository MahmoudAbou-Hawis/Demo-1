Detailed Description

UART (Universal Asynchronous Receiver/Transmitter) interface definitions for STM32F401CC microcontrollers.

Project Name
stm32f4xx drivers
Code Language
C
Description
This header file provides a generic interface for interacting with multiple UART peripherals on the STM32F401CC microcontroller (UART1,UART2 and UART6).
Author
Mahmoud Abou-Hawis
